# Stokmatik Lite

Stokmatik Lite is a free desktop software for tracking product stock and expiration dates.

It is designed for:
- Markets
- Grocery stores
- Pharmacies
- Warehouses
- Small businesses

## Features
- Barcode-based product management
- Stock in / stock out operations
- Expiration date control
- Automatic warning for expired and critical products
- Expiration report screen
- Export to Excel
- Works offline
- No installation required

## How to use
1. Run the exe file.
2. Scan or enter barcode.
3. Enter quantity.
4. Choose SALE or STOCK IN.
5. Click PROCESS.

If product does not exist, product add screen will open automatically.
